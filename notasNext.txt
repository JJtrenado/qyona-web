npm run nos dice los sripts que tenemos en nuestra aplicación
    start ----> inicia nueestra aplicación
    dev ------> modo desarrollo
    build ----> crea los estáticos para deployar nuestra aplicación (deploy: llevar un sitio web a internet)

----------------
inicializamos el proyecto:
    npm run dev
        inicializa el servidor
        crea cosas y aparte crea la carpeta .next (hace un bundle, un build con todos los estáticos del servidor y lo que ahce es que si hacemos un cambio ba a ser super rapido)
